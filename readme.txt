													Wikipedia Search Engine


1) Run search.py : python3 search.py
a) It will take the Index folder by default from current directory, and generate the output in current folder to as output.txt.
b) On Running the code , it will ask for search query and print the output on the Screen.


2) Indexing.py : python3 Indexing.py wiki.xml <Path to folder to Index file>
a) It will create the index files


3) merge.py : python3 merge.py
a) It will merge all the files give as input .
